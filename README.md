# Oyun Çeviri Firması Website

Modern ve profesyonel bir oyun çeviri firması websitesi. Next.js 14, TypeScript ve Tailwind CSS ile geliştirilmiştir.

## 🚀 Başlangıç

### Gereksinimler
- Node.js 18+ 
- npm veya yarn

### Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

3. Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açın.

## 📁 Proje Yapısı

```
anonymous/
├── app/
│   ├── layout.tsx      # Ana layout
│   ├── page.tsx        # Ana sayfa
│   └── globals.css     # Global stiller
├── components/
│   ├── Header.tsx      # Header/Navigation
│   ├── Hero.tsx        # Hero bölümü
│   ├── Services.tsx    # Hizmetler bölümü
│   ├── About.tsx       # Hakkımızda bölümü
│   ├── Contact.tsx     # İletişim formu
│   └── Footer.tsx      # Footer
└── package.json
```
