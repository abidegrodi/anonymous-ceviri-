# Font Dosyaları

Caviar Dreams font dosyalarını bu klasöre ekleyin:

- `CaviarDreams.ttf` veya `CaviarDreams.woff` veya `CaviarDreams.woff2`
- `CaviarDreams-Bold.ttf` veya `CaviarDreams-Bold.woff` veya `CaviarDreams-Bold.woff2`

Font dosyalarını buraya ekledikten sonra, `app/globals.css` dosyasındaki `@font-face` tanımları otomatik olarak çalışacaktır.

Eğer font dosyalarınız farklı isimlerdeyse, `app/globals.css` dosyasındaki font dosya yollarını güncelleyin.
